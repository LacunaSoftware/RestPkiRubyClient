## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rest_pki'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rest_pki

## Usage

    $ require 'rest_pki'

Create a initialize file

    $ RestPki.api_key = 'PLACE YOUR API KEY/ACCESS TOKEN'
